設定や利用方法は書籍を参照してください。

書籍は次の場所で販売中（販売していました）です。

* 技術書典（ https://techbookfest.org/product/5460536974114816 ）
* BOOTH（ https://morinomiyakono.booth.pm/items/3585460 ）
